## Dependencies

You need to have:
* NodeJS >= 5.x. ( better v6.7.0 )

Install global NPM packages Gulp and Bower:

<pre>
sudo npm install bower -g
sudo npm install gulp -g
</pre>

Install project dependencies:
<pre>
npm install
bower install
</pre>

Start dev environment:
<pre>
gulp serve
</pre>
Build for production:
<pre>
gulp build
</pre>
